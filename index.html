Monument Valley is a puzzle game created by UsTwo, available on both iOS and Android.

This project attempts to loosely mimic the text effects from the game and are dynamically generated using WebGL and the Three.js library. The repository is up on GitHub for those who are interested in digging into the code.

It was created by Andrew Cobley, a budding web developer based in Cardiff.
